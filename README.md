# PythonDZ1_2
Домашнее задание 1 Программирование на Python Мухутдинов

## Описание
Программа является прототипом онлайн дейтинг-сервиса, подбирает пары по алфавиту их имен.
В случае несовпадения количества мужчин и женщин выводит ошибку.

## Примеры работы
Входные данные: 
boys = ['Peter', 'Alex', 'John', 'Arthur', 'Richard', 'Максим'] 
girls = ['Снежана', 'Kate', 'Лиза', 'Kira', 'Emma', 'Trisha', ] 

Результат: 
Идеальные пары: 
Alex и Emma 
Arthur и Kate 
John и Kira 
Peter и Trisha 
Richard и Лиза 
Максим и Снежана 

Входные данные: 
boys = ['Peter', 'Alex', 'John', 'Arthur', 'Richard', 'Максим'] 
girls = ['Kate', 'Лиза', 'Kira', 'Emma', 'Trisha',]

Результат: 
Внимание, кто-то может остаться без пары!
